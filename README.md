# RFID-Smart-Lock
Build your own RFID lock and allow only authorized people to access your locker!

## Hardware Required
- Arduino Uno R3
- Relay Module 1 Channel 5V
- RFID / NFC Module PN532 13.56MHz
- Power Adapter 12V 2.5A Safety Mark
- Jumper Wire Male-Male 40x
- Solenoid Door Lock
